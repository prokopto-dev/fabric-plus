Contact¶

If you’re still looking for more information about FabricPlus, you can find the developer on various platforms, or via the repository.

Mastodon¶

You can find the principal developer, @prokoptodev@mastodon.social, on Mastodon; follow for announcements and updates, as well as random posts.

Bugs/Issue Tracker¶

To file new bugs, seach existing ones, or make feature requests, please visit the FabricPlus issue tracker. You will need a GitLab account to do so.

You can also request updates there to the documentation, as well as clarification on documentation points.
